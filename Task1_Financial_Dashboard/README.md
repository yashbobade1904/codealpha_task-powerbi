# Task 1: Financial Health Dashboard

## Objective
Develop a dashboard analyzing SME financial status.

## Dashboard Features
- KPI Cards (Financial Literacy, Reporting, Risk, Analysis)
- SME Size Distribution (Donut Chart)
- Avg Financial Health by Sector (Bar Chart)
- Risk Assessment by SME Size (Bar Chart)
- Financial Health Trend by Year (Line Chart)
- Interactive Slicers (Sector, Size)

## Tool Used
Power BI Desktop

## Dataset
SME Financial Decision Dataset
